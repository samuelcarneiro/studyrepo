#### Tipos de dados
- **Inteiros**
	- int
	- `idade = 30`, `quantidade = 100`
- **Flutuantes**
	- float
	- `preco = 9.90`, `altura = 1.70`
- **Cadeias de texto**
	- string
	- `nome = "Samuel`, `mensagem = 'Olá mundo'`
- **Booleanos**
	- True ou False
	- `é maior de idade = True`, `tem desconto = False`

> ==Em Python os valores booleanos possuem a letra inicial maiúscula.==
